Для использования всех возможностей Фреймворка для каждой информационной базы требуется собственная лицензия.

Лицензия активируется при установке Фреймворка в информационную базу посредством указания кода активации, полученного при покупке.

Действующая лицензия предоставляет доступ к обновлениям компонентов Фреймворка.
